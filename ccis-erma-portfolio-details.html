Project Overview

The CCIS ERMA (Equipment and Resource Management App) is a standalone system designed specifically for managing the inventory of equipment, facilities, and supplies at the College of Computing and Information Sciences (CCIS). It operates independently and does not rely on or integrate with any third-party platforms or external systems. This allows the app to focus solely on addressing the inventory management needs within CCIS.

The system has been successfully rolled out to the administrators and deployed within the CCIS LAN, utilizing Laravel with Blade for the frontend and backend, MySQL for the database, and NGINX as the web server.

Key Features | Modules

Responsive Design

CCIS ERMA is fully responsive layout across devices, accessible and user-friendly on desktops, tablets, and smartphones

User Management

It features user management, allowing registration, login, profile management, and role-based access control (RBAC) for Super Admin, Admin, Staff, and Faculty users.

Resource Management

The system includes dedicated modules for managing equipment, facilities, supplies, and materials, each supporting Excel-based importation, exportation, creation, updating, and deletion of records. The requests and borrowed items management feature enables users to submit and update item requests, track return statuses, and export data as needed.

Basic Dashboard Analytics

A basic analytics dashboard presents key metrics through counts, graphs, and tables, offering a quick overview of system data—such as total equipment, daily borrowed items, and equipment counts categorized by status and person liable, with support for date-based filtering.

Stock Notifications

The system provides notifications for out-of-stock or critical stock items to keep the CCIS procurement officers aware of inventory shortages and potential supply issues.
